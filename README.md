# SC2 API Public Community Guide

Uses MkDocs and Material MkDocs.  

## Installation

```bash
    $ pipenv install
    $ pipenv install mkdocs
    $ pipenv install mkdocs-material
    $ pipenv install mkdocs-material-extensions
    $ pipenv shell
    $ mkdocs serve
```
